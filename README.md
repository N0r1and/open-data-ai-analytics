# open-data-ai-analytics

## Назва і мета
Аналіз індексів цін на житло в Україні на основі відкритих даних Державної служби статистики.  
Мета: дослідити динаміку цін на первинному та вторинному ринках нерухомості, виявити тенденції та аномалії, зокрема вплив зовнішніх факторів (повномасштабне вторгнення 2022 року).

## Джерело даних
- **Датасет:** Індекси цін на житло
- **Джерело:** [data.gov.ua](https://data.gov.ua/dataset/85e5e66c-be8b-4e75-a7df-d3e4c886fa73)
- **Формат:** XLSX
- **Охоплення:** 2016 Q1 — 2025 Q2 (квартальні дані)
- **Показники:** індекси цін на первинному та вторинному ринках (1-, 2-, 3-кімнатні квартири)
